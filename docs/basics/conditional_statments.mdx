---
title: Conditional Statements (if, else)
sidebar_position: 3
---

The `if(...)` instruction only accept **bool** values:
* `true`;
* `false`;

## Examples

Example:
```zig title="~/example/main.zig"
const std = @import("std");

pub fn main() !void {
    const stdout = std.io.getStdOut().writer();

    if (true) {
        try stdout.print("Hello {s}\n", .{"world"});
    } else {
        try stdout.print("It was false :(");
    }
}
```

```bash title="~/example via ↯ v0.11.0 took 2s"
➜ zig run main.zig
Hello world
```

:::caution
Zig does not support truthy or falsy values, it means if you will try to use as example `0` in if statment you will get an error. 
:::

```zig title="~/example/main.zig"
const std = @import("std");

pub fn main() !void {
    const stdout = std.io.getStdOut().writer();

    if (0) {
        try stdout.print("Hello {s}\n", .{"world"});
    }
}
```
